# e-plantShopping (Paradise Nursery)

## Project Overview
This project is a React-based e-commerce application for a plant nursery called "Paradise Nursery". It allows users to browse various houseplants, add them to a shopping cart, and manage their orders.

## Features
- **Landing Page:** A welcoming interface with a "Get Started" button.
- **Product Listing:** Display of houseplants with images, names, and prices, organized by categories.
- **Shopping Cart:** Functional cart with increment/decrement features and total cost calculation.
- **Redux Integration:** State management for cart operations.

## Technologies Used
- React JS
- Redux Toolkit
- React Router
- CSS

## How to Run
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm start` to launch the application.
